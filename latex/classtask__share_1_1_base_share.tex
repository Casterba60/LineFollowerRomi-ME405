\doxysubsection{Detailed Description}
Base class for queues and shares which exchange data between tasks. 

One should never create an object from this class; it doesn\textquotesingle{}t do anything useful. It exists to implement things which are common between its child classes {\ttfamily \doxylink{classtask__share_1_1_queue}{Queue}} and {\ttfamily \doxylink{classtask__share_1_1_share}{Share}}.
